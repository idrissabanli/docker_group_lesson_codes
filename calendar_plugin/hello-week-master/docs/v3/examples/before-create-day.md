# Before Create Day

A hook for modifying a day DOM.

## Usage

```js
new HelloWeek({
  selector: '.calendar',
  beforeCreateDay: data => {
    return { 
      ...data, 
      ... {
        node: el('div', data.attributes, el('span', {}, data.day.toString()))
      }
    }
  }
});
```
